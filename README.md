# BookStoreApp
Trabalho Prático de Teste de Software
Lucas de Oliveira Ferreira

O sistema desenvolvido será uma aplicação de linha de comando chamada BookStoreApp, que simula o funcionamento básico de uma livraria virtual. O usuário poderá cadastrar livros, listar o catálogo, buscar livros por título e registrar vendas.

O sistema mantém os dados em memória, o que permite fácil execução e foco na lógica e nos testes, sem necessidade de um banco de dados.
